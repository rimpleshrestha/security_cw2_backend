# SkinMuse Backend

This is the backend for the SkinMuse web application, built using **Node.js**, **Express**, and **MongoDB**.

---

## Features
- User authentication (signup, login)
- Post CRUD operations (create, read, update, delete)
- Comment CRUD operations (create, read, update, delete)
- User profile management (update details, profile image, change password)
- JWT authentication and authorization

---

## Tech Stack
- **Node.js** with **Express**
- **MongoDB** with Mongoose
- **JWT** for authentication
- **Multer** for file uploads
- **Jest + Supertest** for testing

---

## Requirements
- Node.js (>= 16)
- MongoDB (local or Atlas)
